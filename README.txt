homer: random poetry to stdout, only for UNIX-like systems

usage:
	$ homer 
	$ homer n [where n is the number of txt files in .poem]

installing:
	$ make
	$ make install

note:
	run only from ~/ after installing
	this code is under the public domain

	the ~/.poem directory should contain your text files 
	and it should be numbered in series so the program can 
	randomly select the files 

	by default this program has the verses from bhagawad gita
